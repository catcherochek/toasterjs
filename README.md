# toasterjs

##toaster messages for your  frontend*


![image](success.png)


**Simply add this lib  with**

` <script src="toaster.js"></script>`

**And then in script create an object**

` <script>
    var toaster = new Toaster();
  </script>`
  
  **After that you can call**
  
  `toaster.theToast(message,type)`
  
  **type**
  MSG_ERROR = 1;
  MSG_WARNING = 2;
  MSG_SUCCESS = 3;
  MSG_INFO = 4;
